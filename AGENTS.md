# Agent Instructions

This project uses **git** for version control.

## Version Control

Standard git commands:
- `git status` - show working tree status
- `git diff` - show changes
- `git log` - show commit history
- `git add` / `git commit` - stage and commit changes

## Issue Tracking

This project uses `bd` (beads) for issue tracking. Issues are stored in `.beads/issues.jsonl`.

## Code Style

- Elixir project - follow `.claude/ELIXIR_STYLE.md`
- Always run `mix format` before commits
- Run `mix test` to verify changes
